# 📊 Dashboard de Análise de Vendas

## 📌 Sobre o Projeto

Este projeto tem como objetivo analisar dados de vendas de uma empresa fictícia, utilizando SQL para manipulação de dados e Power BI para visualização.

---

## 🛠️ Ferramentas Utilizadas

* SQL (MySQL)
* Power BI
---

## 📈 Análises Realizadas

* Faturamento total
* Total de pedidos
* Ticket médio
* Faturamento por categoria
* Evolução do faturamento ao longo do tempo
* Top 5 produtos mais vendidos
* Top 5 clientes que mais compraram
* Distribuição de pedidos por estado

---

## 🧠 Insights

* A categoria de eletrônicos concentra a maior parte do faturamento
* Existe variação significativa de vendas ao longo dos meses
* Alguns clientes concentram grande parte das compras

---

## 📂 Estrutura dos Dados

Os dados foram modelados em um banco relacional contendo as seguintes tabelas:

* clientes
* pedidos
* produtos
* itens_pedido

---

## 🚀 Objetivo

Este projeto foi desenvolvido para prática e demonstração de habilidades em análise de dados, modelagem e visualização.
